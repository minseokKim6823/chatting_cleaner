# Android Auto Reply App

이 앱은 카카오톡 알림을 읽어서 Spring 서버에 판정 요청을 보내고, 위반이면 카카오톡 알림의 답장 액션으로 자동 경고 메시지를 보냅니다.

## 역할

- 카카오톡 알림 수신
- 방 이름, 보낸 사람, 메시지 추출
- 서버 `/api/bridge/auto-reply` 호출
- `shouldReply=true` 이면 자동 답장 전송

## 필요한 것

- 안드로이드 8.0 이상 권장
- 카카오톡 설치
- 알림 접근 권한 허용
- 같은 네트워크에서 접근 가능한 서버 URL

## 빌드

Android Studio에서 `android-app` 폴더를 별도 프로젝트로 열고 `app` 모듈을 빌드합니다.

APK 기본 산출물:

- `app/build/outputs/apk/debug/app-debug.apk`

## 설치 후 설정

1. 앱 실행
2. `서버 URL` 입력
3. `허용 방` 입력
4. `봇 이름` 입력
5. 저장
6. `알림 접근 설정 열기`로 권한 허용

## 주의

- 카카오 공식 API가 아니라 알림 기반 자동응답 방식입니다.
- 카카오톡 알림 형식이 바뀌면 방/보낸 사람 파싱이 깨질 수 있습니다.
